# 1か月単位の変形労働時間制 シフト管理システム 要件定義書

## 1. 目的

本システムは、1か月単位の変形労働時間制を適切に運用するためのシフト管理システムである。

主な目的は以下のとおり。

- 労働日および日ごとの労働時間を、対象期間開始前に明確に特定する
- 1日、1週間、対象期間全体の労働時間を自動集計する
- 法定労働時間の総枠を自動算出し、上限超過を検知する
- 年少者、妊産婦、育児・介護等を行う労働者への配慮事項を管理する
- 割増賃金計算に必要な残業時間を、1日・1週間・対象期間全体の3段階で判定する
- 労働基準監督署への届出、職場掲示、給与計算連携に使えるデータを出力する

## 2. アーキテクチャとの対応

本要件は `Architecture.md` の構成を前提とする。

### 2.1 Frontend

- 管理者ダッシュボード
  - シフト作成、編集、確定、警告確認を行う
- 従業員用シフト確認画面
  - 確定済みシフト、勤務予定、必要な配慮事項を確認する
- 設定・マスタ管理画面
  - 事業場設定、対象期間、起算日、従業員属性、配慮事項を管理する

### 2.2 Backend Logic

- シフト作成・編集エンジン
  - シフト入力、労働時間算出、確定、ロック制御を担う
- コンプライアンス・チェック
  - 年少者、妊産婦、育児・介護等の配慮対象者に対する制限・警告を担う
- 労働時間計算エンジン
  - 日別、週別、対象期間別の所定労働時間を集計する
- 割増賃金・残業判定エンジン
  - 1日単位、1週間単位、対象期間全体の3段階で超過時間を判定する

### 2.3 Database

- 従業員マスタ
  - 従業員属性、制度対象可否、配慮事項を保持する
- カレンダー・法定上限マスタ
  - 対象期間、起算日、週法定労働時間、暦日数、上限時間を保持する
- 確定シフト・実績データ
  - シフト予定、確定状態、実績、集計結果、残業判定結果を保持する

## 3. 用語

| 用語 | 意味 |
| --- | --- |
| 対象期間 | 1か月以内で設定される変形労働時間制の集計期間 |
| 起算日 | 対象期間の開始日 |
| 週法定労働時間 | 原則40時間。特例措置対象事業場では44時間を選択可能 |
| 法定労働時間の総枠 | 対象期間の暦日数に応じて算出される労働時間の上限 |
| 所定労働時間 | シフト上、会社があらかじめ定めた労働時間 |
| 法定内残業 | 所定労働時間を超えるが、法定労働時間の範囲内に収まる時間 |
| 法定外残業 | 法定労働時間を超え、割増賃金の判定対象となる時間 |

## 4. 機能要件

### REQ-001 事業場設定

事業場ごとに、1週間の法定労働時間を設定できること。

- 原則は40時間とする
- 特例措置対象事業場の場合は44時間を選択できること
- 44時間を選択する場合、対象事業場であることを識別できる項目を持つこと

### REQ-002 対象期間・起算日設定

管理者は、1か月以内の対象期間と起算日を設定できること。

- 対象期間は暦日数で管理する
- 対象期間ごとに週法定労働時間を紐づける
- 対象期間確定後は、既存シフトへの影響を確認できること

### REQ-003 従業員マスタ管理

従業員ごとに、制度適用や配慮に必要な属性を管理できること。

- 制度対象者かどうか
- 18歳未満の年少者かどうか
- 妊産婦であり、請求により制限対象となるかどうか
- 育児・介護等により配慮が必要かどうか
- その他、勤務制限や注意事項

### REQ-004 シフト入力

管理者は、対象期間内の労働日ごとに勤務予定を入力できること。

- 勤務日
- 始業時刻
- 終業時刻
- 休憩時間
- 日別の所定労働時間
- 備考、配慮事項

### REQ-005 労働時間の自動計算

システムは、入力されたシフトから労働時間を自動計算すること。

- 日別労働時間
- 週別労働時間
- 対象期間全体の労働時間
- 所定労働時間、法定内残業時間、法定外残業時間の区分

### REQ-006 法定労働時間の総枠算出

システムは、対象期間の暦日数に応じて法定労働時間の総枠を自動算出すること。

計算式:

```text
法定労働時間の総枠 = 週法定労働時間 × 対象期間の暦日数 ÷ 7
```

例:

```text
週法定労働時間40時間、対象期間30日の場合
40 × 30 ÷ 7 = 171.4時間
```

### REQ-007 上限超過アラート

対象期間内の所定労働時間の合計が法定労働時間の総枠を超える場合、警告を表示すること。

- シフト作成中に警告する
- シフト確定前に警告する
- 警告対象の従業員、日、週、対象期間を確認できること

### REQ-008 シフト確定とロック

管理者は、対象期間のシフトを確定できること。

- 確定後は、原則として会社側から任意に労働時間を変更できないようにする
- 変更が必要な場合は、変更理由、変更者、変更日時を記録する
- 確定前、確定後、変更済みなどの状態を管理する

### REQ-009 コンプライアンス・チェック

システムは、シフト入力時および確定時にコンプライアンスチェックを行うこと。

- 年少者に対して、1日8時間・週40時間を超えるシフトが組まれていないか確認する
- 妊産婦から請求がある場合、制限対象となるシフトが組まれていないか確認する
- 育児・介護等を行う労働者について、必要な配慮事項を確認できるようにする
- チェック結果は、警告またはエラーとして管理者に表示する

### REQ-010 割増賃金・残業判定

システムは、割増賃金計算に必要な法定外残業時間を3段階で判定すること。

1. 1日単位の判定
   - 8時間を超える時間を設定した日は、その設定時間を超えた分を抽出する
   - 上記以外の日は、8時間を超えた分を抽出する

2. 1週間単位の判定
   - 40時間または44時間を超える時間を設定した週は、その設定時間を超えた分を抽出する
   - 上記以外の週は、40時間または44時間を超えた分を抽出する
   - 1日単位で抽出済みの時間は重複して抽出しない

3. 対象期間全体の判定
   - 法定労働時間の総枠を超えて労働した時間を抽出する
   - 1日単位、1週間単位で抽出済みの時間は重複して抽出しない

### REQ-011 シフト表出力

システムは、確定シフトを出力できること。

- 労働基準監督署への届出に利用できる形式
- 職場掲示に利用できる形式
- 従業員ごとの確認に利用できる形式
- 対象期間、起算日、事業場設定、日別勤務予定を含めること

### REQ-012 賃金計算連携

システムは、給与計算に必要な集計データを出力できること。

- 従業員別の所定労働時間
- 従業員別の法定内残業時間
- 従業員別の法定外残業時間
- 日別、週別、対象期間別の集計根拠
- 給与システム等へ連携しやすいCSV形式またはAPI形式

## 5. 主要データ

### 5.1 従業員

- 従業員ID
- 氏名
- 所属
- 雇用区分
- 制度対象フラグ
- 年少者フラグ
- 妊産婦制限対象フラグ
- 育児・介護等配慮対象フラグ
- 配慮事項メモ

### 5.2 対象期間設定

- 対象期間ID
- 事業場ID
- 起算日
- 終了日
- 暦日数
- 週法定労働時間
- 法定労働時間の総枠

### 5.3 シフト

- シフトID
- 対象期間ID
- 従業員ID
- 勤務日
- 始業時刻
- 終業時刻
- 休憩時間
- 所定労働時間
- 確定状態
- 変更理由
- 変更者
- 変更日時

### 5.4 集計結果

- 集計ID
- 対象期間ID
- 従業員ID
- 日別労働時間
- 週別労働時間
- 対象期間合計労働時間
- 法定内残業時間
- 法定外残業時間
- 上限超過フラグ
- コンプライアンス警告内容

## 6. 画面要件

### 6.1 管理者ダッシュボード

- 対象期間ごとのシフト状況を確認できること
- 上限超過、コンプライアンス警告、未確定シフトを一覧で確認できること
- シフト確定、出力、集計再計算を実行できること

### 6.2 シフト作成・編集画面

- 従業員別、日別にシフトを入力できること
- 入力中に労働時間を自動計算すること
- 上限超過や制限対象者への警告を表示すること
- 確定済みシフトはロック状態として表示すること

### 6.3 従業員用シフト確認画面

- 従業員が自分の確定シフトを確認できること
- 勤務日、始業時刻、終業時刻、休憩時間を確認できること
- 必要に応じて配慮事項や注意事項を確認できること

### 6.4 設定・マスタ管理画面

- 事業場設定を管理できること
- 対象期間と起算日を管理できること
- 従業員属性と配慮事項を管理できること
- カレンダーと法定上限マスタを管理できること

## 7. 非機能要件

- 計算結果の根拠を後から確認できること
- 確定済みシフトや変更履歴を監査できること
- 権限により、管理者と従業員の操作範囲を分離すること
- 法令解釈が変わる可能性のある設定値は、コード固定ではなくマスタ管理を優先すること
- 給与連携データは再出力できること

## 8. 未決事項・確認事項

- 44時間制を適用できる事業場の判定条件を、どの粒度でマスタ化するか
- 妊産婦の「請求時」フラグを誰が、どのタイミングで登録するか
- 育児・介護等を行う労働者への配慮を、警告に留めるか、シフト確定不可にするか
- シフト確定後の変更を認める業務フローと承認権限
- 労働時間の丸め単位、休憩控除、日跨ぎ勤務の扱い
- CSV出力項目と給与システム側の受け入れ形式
- 法令要件の最終確認を、社労士または法務担当者が行うタイミング
